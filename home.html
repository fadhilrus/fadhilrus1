PROJECT

Project Mykesnik, merupakan project web aplikasi saya yang kedua, My Kesnik berguna untuk informasi dan pendaftaran klinik kesehatan dan digunakan unutuk mengelola atau mencatat daftar jadwal praktek dokter dan daftar pengobatan. Tujuan dibuatnya project ini karena persyaratan tugas pada semester 4 dan saya juga mendapatkan ide yang berawal dari kunjungan saya untuk berobat ke klinik terdekat dirumah saya, namun pendaftaran disana harus offline, lalu dibuatlah web aplikasi yang berguna untuk memudahkan informasi agar para pasien dapat lebih fleksibel ketika ingin berobat. Mykesnik merupakan ide saya sendiri dan memiliki struktur HTML, CSS, JS yang terkoneksi oleh PHP sebagai database untuk menyimpan segala hal yang diperlukan dalam web aplikasi tersebut. Mykesnik belum pernah dipakai dalam kegiatan sehari-hari karena masih dalam tahap pengembangan oleh saya sendiri.
Project INVENTY, merupakan project pertama kali saya dan teman kelompok saya. Namun pada akhirnya saya sendiri yang mengerjakan karena ada beberapa hal diluar kendali yang terjadi, hal itu membuat saya menjadi lebih bisa belajar dalam pembuatan web aplikasi ini. INVENTY berfungsi sebagai web aplikasi untuk inventaris barang, anda dapat menginput barang barang dan mendata barang apa saja yang sudah anda punya. Biasanya web ini berguna pada perusahaan pabrik atau retail sebagai pendataan barang masuk dan keluar. Project ini saya kerjakan pada semester 2 dan lumayan membuat saya pusing karena harus mengulik front end sampai dengan back end.
